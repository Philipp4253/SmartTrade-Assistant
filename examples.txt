# advice prompt
Дай краткий анализ рыночной ситуации: 1) Суть, 2) Торговая идея (лонг/шорт), 3) Уровни (TP/SL), 4) Риск на сделку.

# analyze prompt
Анализируй этот текст: <текст>. Выдели: ключевые факты, влияние на рынок, возможные торговые действия.

# strategy prompt
Составь простой чек-лист для входа в сделку (триггер, подтверждение, объем, стоп).
